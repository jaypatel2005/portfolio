# Jay Patel Portfolio

Static personal website built for deployment on GitHub Pages and compatible with subdirectory hosting such as an IIT Bombay CSE personal web space.

## Local preview
Open `index.html` in a browser, or run a simple local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Important
- Replace `assets/profile.jpg` with a higher-resolution portrait when available.
- All internal asset paths are relative for compatibility with both GitHub Pages and `~username/` hosting.
